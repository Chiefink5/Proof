Chore Logger MVP

Files:
- index.html
- styles.css
- app.js
- manifest.json
- sw.js

How to use:
1. Open index.html in a browser.
2. Add groups and sub-chores in Manage.
3. Use + or - on the Home screen to log actions.
4. Use Manual Entry for backdated logs or edits.
5. Use Log to filter and review history.

Notes:
- Data is stored locally in browser localStorage.
- If you move browsers/devices, the data will not move with it unless you add a backend later.
- PWA install support is included, but icons are not yet included.
